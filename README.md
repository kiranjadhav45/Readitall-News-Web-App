## Run Locally

Clone the project

```bash
  git clone https://github.com/kiranjadhav45/Readitall-News-Web-App.git
```

Go to the project directory

```bash
  cd Readitall-News-Web-App
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
